### Question 7955

### Question 7932
> Возач је:
> - [ ] лице које по путу сопственом снагом гура или вуче возило
> - [ ] свако лице које има возачку дозволу
> - [x] **лице које на путу управља возилом**

**Comment:**
> По [определению из закона](zakon?chapter=II&chlan=7&paragraph=79), водитель — это лицо, которое управляет транспортным средством на дороге.
>
> Человек, толкающий автомобиль, не считается водителем. Пешеход, у которого в кармане лежит водительское удостоверение, тоже не является водителем.

### Question 7936
> Означити неопходне услове који морају бити истовремено испуњени да би се сматрало да је возило заустављено (multi-select):
> - [x] прекид кретања возила није наступио услед поступања по знаку или правилу
> - [ ] возач је изашао из возила али се од истог није удаљио више од 5 m
> - [x] возач није напустио возило
> - [x] прекид кретања возила на путу траје до три минута
> - [ ] мотор возила је искључен
> - [ ] у возилу нема ниједног лица

**Comment:**
> Условия _заустављања_ (остановки) _возила_:
>
> * движение _возила_ прекращено
> * водитель не покинул _возило_
> * продолжительность до трех минут
> * это не произошло из-за знака или правила, регулирующего дорожное движение
>
> Это следует из [определения _заустављања возила_ из закона](zakon?chlan=7&paragraph=82).
>
> Таким образом:
> * _прекид кретања возила није наступио…_ — вариант подходит под определение
> * _возач је изашао из возила…_ — вариант не подходит, так как водитель вышел
> * _прекид кретања возила на путу траје до три минута_ — вариант подходит под определение
> * _мотор возила је искључен_ — не подходит, так как при остановке мотор может быть и включен
> * _у возилу нема ниједног лица_ — не подходит, так как водитель вышел

*Shape: definition breakdown as a bullet list, then every choice re-checked against each bullet one by one. This is the pattern for multi-select "which conditions must ALL hold" questions.*

### Question 7958
> Који од набројаних услова морају бити истовремено испуњени да би се сматрало да је на путу колона возила (multi-select):
> - [x] начин кретања возила је међусобно условљен и између њих не може без ометања ући друго возило
> - [ ] возила су паркирана у истој саобраћајној траци
> - [x] низ од најмање три возила
> - [ ] возила се крећу на растојању које омогућава улазак другог возила без ометања осталих возила
> - [ ] низ од најмање два возила
> - [x] возила се крећу једно иза другог истом саобраћајном траком у истом смеру

**Comment:**
> В [определении _колони возила_ из закона](zakon?chlan=7&paragraph=88) установлено, что колонна:
>
> * состоит из не менее **трех** транспортных средств
> * они **остановлены** или движутся в одном направлении
> * расположены друг за другом на одной полосе движения
> * способ движения ТС взаимосвязан
> * между ТС не может без помех въехать другое транспортное средство
>
> Таким образом, подходят следующие варианты:
>
> * _начин кретања возила је међусобно условљен и између њих не може без ометања ући друго возило_
> * _низ од најмање три возила_
> * _возила се крећу једно иза другог истом саобраћајном траком у истом смеру_
>
> Варианты, которые не подходят:
>
> * _возила су паркирана у истој саобраћајној траци_ — не подходит, так как транспортные средства в этом варианте припаркованы, а не остановлены.
> * _низ од најмање два возила_ — не подходит, так как в определении указано, что колонна состоит из не менее трех транспортных средств
> * _возила се крећу на растојању које омогућава улазак другог возила без ометања осталих возила_ — не подходит, так как в определении указано, что между ТС не может без помех въехать другое транспортное средство.

*Shape: same "definition -> bullet -> sort choices into two buckets" pattern as 7936, reused for a different definition (chlan=7, paragraph=88 = "колона возила"). Note: `comments_cli.py search-law колона` would have surfaced this same paragraph directly.*

### Question 7931
> Шта се од набројаног сматра саобраћајем? (multi-select)
> - [ ] Кретање возила полигоном за пробне вожње или спортске приредбе
> - [x] Гурање бицикла пешачком стазом
> - [ ] Кретање лица по тргу
> - [x] Кретање бицикла бициклистичком стазом
> - [x] Кретање мотокултиватора земљаним путем
> - [ ] Гурање возила по платоу намењеном кретању и окупљању лица

**Comment:**
> Чтобы отвечать на подобные вопросы, можно запомнить: саобраћај — это **движение** (_кретање_) по **дороге** (_путу_), улице или тротуару, а также пешеходным и велосипедным дорожкам. Движение по частным территориям, полям, полигонам и т.п. не считается _саобраћајем_.
>
> *Саобраћајем* (дорожным движением) из перечисленных вариантов считается:
>
> * *Гурање бицикла пешачком стазом*, так как это дорога для пешеходов по определению, движение по ней регулируется
> * *Кретање бицикла бициклистичком стазом*, так как велосипедная дорожка — тоже дорога по определению, на которой движение регулируется
> * *Кретање мотокултиватора земљаним путем*, так как грунтовая дорога — дорога, на которой движение регулируется
>
> Не является *саобраћајем*:
> * *Кретање возила полигоном за пробне вожње или спортске приредбе* — движение по частной территории
> * *Кретање лица по тргу*, так как площадь — это не дорога и не часть дороги
> * *Гурање возила по платоу намењеном кретању и окупљању лица* — так как подобные площадки не являются дорогой.
>
> Вышеперечисленное следует из закона, в том числе из:
>
> * [определения _саобраћаја_](https://saobracaj.app/zakon?chapter=II&chlan=7&paragraph=2)
> * [определения _земљаног пута_](https://saobracaj.app/zakon?chapter=II&chlan=7&paragraph=10)
> * [определения _трга_](https://saobracaj.app/zakon?chapter=II&chlan=7&paragraph=29)
> * [определения _бициклистичкој стази_](https://saobracaj.app/zakon?chapter=II&chlan=7&paragraph=27)
> * [определения _пешачкој стази_](https://saobracaj.app/zakon?chapter=II&chlan=7&paragraph=26)
